Amy Trading Academy

Website ini dibuat sebagai buku online trading yang ringan, clean, dan langsung bisa dipasang di GitHub Pages.

Materi disusun dari sumber yang kamu kirim: Materi-tradingamy.zip dan kumpulan PDF batch tentang basic trading, ICT, SMC, XAUUSD, risk, psikologi, backtesting, dan journaling.

Format: HTML langsung, tiap materi halaman sendiri, prompt gambar tersedia di tiap materi, dan akses dibatasi memakai kode.

Tentang proteksi kode

Kode akses dibuat dari GitHub Actions. Website hanya membaca hash kode dari file data/access-codes.json. Kode asli tidak ditaruh di halaman website.

Karena GitHub Pages adalah static hosting, sistem ini cocok untuk seleksi akses normal. Untuk proteksi konten yang benar-benar anti-bocor, perlu backend/private hosting atau sistem enkripsi konten.
